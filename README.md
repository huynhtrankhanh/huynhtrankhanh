I used to be working on Contrasleuth, a project that is conjectured to be able to create a mesh network of mobile Android devices. My conjecture has been proven wrong because the discoverServices API is heavily rate limited (my intention was to abuse that API to transmit QUIC packets), the Wi-Fi Direct API requires user interaction and Bluetooth is way too slow as its intended use case is to transmit small amounts of data without draining battery.

I have been infatuated with formal verification since I was first introduced to programming (that is, in 2014) and I think now it's time to act on that desire. I'm now totally committed to learning Lean, the addictive and approachable theorem prover. I believe automated reasoning can significantly transform programming, mathematics and education. The rapid feedback loop that formal verification provides can be highly addictive and gratifying.

I don't have a fixed long term plan as life is unpredictable. But wherever life takes me, I want to work on things that tickle my fancy and refrain from doing things I consider to be immoral.
